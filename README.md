## Adding Your Images

The website is set up with placeholders for the following images:

1. **hello-doggo-logo.jpg** - Your band logo/photo (displayed at the top)
2. **better-times.jpg** - Album cover for "Better Times"
3. **whiskey-lullabies.jpg** - Album cover for "Whiskey & Lullabies"

Simply add these images to the `images/` directory and they will automatically appear on the website.

See `images/README.md` for detailed requirements.
