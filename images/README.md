# Image Placeholders

Please add your images to this directory with the following filenames:

## Required Images:

1. **hello-doggo-logo.jpg** - The Hello Doggo logo/band photo
   - Recommended size: 400x400px (square)
   - This will be displayed as a circular logo at the top of the page

2. **better-times.jpg** - Album cover for "Better Times"
   - Recommended size: 600x600px (square)
   - Standard album cover dimensions

3. **whiskey-lullabies.jpg** - Album cover for "Whiskey & Lullabies"
   - Recommended size: 600x600px (square)
   - Standard album cover dimensions

## Notes:
- Images should be in JPG format
- Keep file sizes reasonable for web (under 500KB each)
- The website will automatically scale images to fit the design
